## Prerequirement
- docker and docker composer
- prisma cli
- nodejs

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```

## Generate Prisma ORM

```sh
npm run prisma
```

## Database Seeder

```sh
npm run db-seed
```
