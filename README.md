## Prerequirement
- docker and docker composer
- nodejs and npm/yarn
- globally node modules for (prisma@1.28.3, typescript, ts-node, tslint, and jest)

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```

## Generate Prisma ORM

```sh
npm run prisma
```

## Database Seeder

```sh
npm run db-seed
```
